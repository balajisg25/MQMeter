LoadRunner Test Analysis and Performance Reporting

Objective: Analyze the attached LoadRunner response time results table along with the param sheet and generate a comprehensive HTML performance summary report.

Requirements:

1. SLA Compliance Check Table

Build the SLA Compliance Table with only these columns:

Transaction Name (if Transaction Type = main)

Sub Transaction Name (if not main)

URL

Min Response Time

Max Response Time

Avg Response Time

90th Percentile Response Time

SLA Status ("Under SLA" or "SLA Breached")


SLA target = 3 seconds (unless specified).

Make the table filterable/searchable in the HTML report.



2. Top 5 Slowest Transactions

List the top 5 transactions with the highest 90th percentile response times.

Include Transaction Name, Sub Transaction Name, 90th Response Time, and SLA Status.



3. Graph

Create a bar chart of the Top 5 slowest transactions by 90th percentile response time.

Y-axis: Response time (seconds)

X-axis: Transaction Name (or Sub Transaction Name where applicable).



4. HTML Report

Sections:

Executive Summary

SLA Compliance Table (with filter/search)

Top 5 Response Time Table

Bar Chart (embedded via base64 or Chart.js)

Observations/Recommendations
